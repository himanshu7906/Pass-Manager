# 🔐 Password Manager

A secure, full-stack password manager built with the MERN stack (MongoDB, Express, React, Node.js). Store, manage, and retrieve your passwords safely.

## ✨ Features

- 🔑 **Secure Storage** — Passwords encrypted before storing in MongoDB
- ➕ **Add / Edit / Delete** — Full CRUD operations for credentials
- 🔍 **Search & Filter** — Quickly find saved passwords
- 📋 **One-click Copy** — Copy passwords to clipboard instantly
- 🔒 **Authentication** — Protected with JWT user login

## 🗂️ Project Structure

```
Pass-Manager/
├── backend/     # Express + MongoDB API
└── src/         # React frontend (Vite)
```

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| **Frontend** | React, Vite, JavaScript |
| **Backend** | Node.js, Express |
| **Database** | MongoDB |
| **Auth** | JWT |

## 🚀 Getting Started

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
npm install
npm run dev
```
